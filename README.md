# D365_FO-xpp-roadmap
Repositorio de práctica X++ para Dynamics 365 Finance & Operations.

## Orden recomendado
1. 01-basics
2. 02-classes
3. 03-data-access
4. 04-extensions
5. 05-transactions
6. 06-testing

## Notas
- Uso de objetos estándar: CustTable, SalesTable, PurchTable, InventTable.
- Algunos ejercicios son demostrativos y pueden requerir adaptar datos reales del entorno.

- Para ejecutar:
  - Crear una clase runnable
  - Compilar el proyecto
  - Ejecuta desde Visual Studio o menú item si corresponde

## Objetivos
- Repasar sintaxis X++
- Entender selects y buffers
- Aprender extensibilidad moderna
- Practicar transacciones y rendimiento
- Empezar con SysTest
